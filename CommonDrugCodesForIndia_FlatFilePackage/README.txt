***************************************************************************
	What is Common Drug Codes for India (Flat File Package) 
***************************************************************************

The Common Drug Codes for India (Flat Files Package) is a managed list of existing SNOMED CT and National Extension and covers unique identifiers, fully specified names (FSN), preferred terms, and status flags.

The package includes files with Tab Separated Values (TSV) file format. 


*****************************************************************
   		       Folder Structure
*****************************************************************

The CommonDrugCodesForIndia_FlatFilePackage.zip contains 07 files.

        SubstanceMaster.txt
        GenericMaster.txt
        BrandMaster.txt
        ProductMaster.txt
        SupplierMaster.txt
        DrugFormMaster.txt
        RouteOfAdministrationMaster.txt
